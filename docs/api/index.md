# API Reference

Full reference for every public module, class, and function in Shapix.

## Modules

| Module | Description |
|--------|-------------|
| **[`shapix`](shapix.md)** | Core module — dimensions, `check`, `check_context`, `make_array_type`. |
| **[`shapix.numpy`](numpy.md)** | NumPy array types, ScalarLike types, and ArrayLike. |
| **[`shapix.jax`](jax.md)** | JAX array types, Tree with `jax.tree_util`, and Like types. |
| **[`shapix.torch`](torch.md)** | PyTorch tensor types and Like types. |
| **[`shapix.optree`](optree.md)** | Tree annotations backed by `optree`. |
| **[`shapix.claw`](claw.md)** | Import hook for package-wide instrumentation. |
